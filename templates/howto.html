Minecraft API Help

Not sure how to use it? You've come to the right place!

Requirements

To get started, you need to add the following line to your HTML file. This is allows for you to easily get data without having to use jQuery or writing your own code to pull the data. 

<script src="//api.syfaro.net/scripts/minecraft.min.js"></script>
(there's even an unminified version if that's your kind of thing)

Basic Usage

It's fairly easy to get data! I'm assuming you know at least a little bit of JavaScript for this next bit. 

<script>
MinecraftAPI.getVersions(function(err, versions) {
    if(err) {
      // handle error
      return;
    }

    // handle data
});
</script>

You can also get the status of a Minecraft server! 

<script>
MinecraftAPI.getServerStatus('ip', { // set ip to your server's ip
    port: 25565, // optional
    favicon: true, // optional
    players: true // optional
}, function(err, status) {
    if(err) {
        // handle error
        return;
    }

    // handle data
});
</script>
